# Intro web

Un petit cours pour apprendre à construire une page web, la publier, et la connecter à une base de données.

## Au programme

- **Git et GitHub** : suivre les versions de son code et le publier.
- **HTML, CSS et JavaScript** : construire une page et la rendre interactive.
- **Supabase** : lire et écrire dans une base de données depuis la page.

## Pour commencer

- Ouvre `site/index.html` dans ton navigateur.
- Commence par l'**Accueil**, puis la page **Cours** (interactive), puis les **briefs** dans l'ordre.

## Contenu

- **Accueil** : la présentation de la semaine.
- **Cours** : JavaScript et le DOM, avec des exemples de code à exécuter et modifier.
- **Briefs** : les exercices, un par jour.

## Astuce

- Certains exemples du cours chargent des images en ligne. S'ils ne s'affichent pas en ouvrant le fichier directement, sers le dossier avec un petit serveur local :

```
python3 -m http.server -d site 8000
```

puis ouvre <http://localhost:8000>.
